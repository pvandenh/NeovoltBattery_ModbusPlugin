# NeovoltBattery_ModbusPlugin
Custom Integration for Neovolt Battery Inverter, using MODBUS connection

Add "https://github.com/pvandenh/NeovoltBattery_ModbusPlugin" as a Custom repo through HACS (select as Integration), then head to Settings > Devices & Services > Add Integration.
Search for "Neovolt Inverter".

Then fill out the integration prompt.
